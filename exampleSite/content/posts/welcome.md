**Zero Waste** is a minimal Hugo theme built around an editorial *paper & ink*
look: warm ivory, ink text, a single sage accent, and IBM Plex typography.

## Code looks good

```python
def fib():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b
```

Hover a code block to reveal the **copy** button. Syntax highlighting is provided
by Chroma; the panel uses the theme's own background token.
